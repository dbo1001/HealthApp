Hola y gracias por usar HealthApp como tu sistema de recomendació nutricional.

Para el correcto uso, seleccione el archivo: HealthApp.exe, haga click derecho sobre el y seleccione: "Crear acceso directo".
Una vez creado el acceso directo desplacelo a donde se prefiera.
Para la ejecución del programa: Haga doble click izquierdo sobre el acceso directo.
Ante cualquier duda, no dude en consultar el manual de Usuario.
