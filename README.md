# HealthApp
Sistema de recomendación nutricional, basado en el auto-aprendizaje por parte del usuario y teniendo en cuenta las diferentes patologías de este para una mejor recomendación.<br/>
Se crea un sistema de recomendación único, con un innovador cálculo de la tasa metabólica Basal, reajustandose en base a la experiencia del usuario.
## Instrucciones
********Desarrolladores:******** <br/>
Si se desea seguir con el desarrollo y mejora del proyecto se encontrará una breve explicación de las instrucciones a seguir en el directorio ForDevelopments. Además de el archivo requeriments.txt, para la instalación de las librerías necesarias. El cual se ha de actualizar con el uso de nuevos librerías.<br/>
Si se desea una explicación más detalla dentro del directorio memorias, se encontrará un anexo, con el manual del desarrollador extendido.<br/>
********Usuarios:********<br/>
Para el uso del programa a nivel usuario, encontraran el ejecutable en la carpeta ForUsers/HealthApp. El archivo: HealthApp.exe, ya esta listo para su ejecución sin una previa instalación.<br/>
Ante cualquier duda se dispondrá de una breve serie de instrucciones pre-uso de la aplicación y de un manual extendido, donde se explican las diferentes funciones y la navegabilidad del programa.<br/>
