###############################
#####-     HEALTHAPP      -####
###############################
Sistema de recomendación nutricional, basado en el auto-aprendizaje por parte del usuario y teniendo en cuenta las diferentes patologías de este para una mejor recomendación.
Se crea un sistema de recomendación único, con un innovador cálculo de la tasa metabólica Basal, reajustandose en base a la experiencia del usuario. Desarrollado en Python


##################################
# Descripción de los directorios #
##################################
ForDevelopments:
Carpeta donde se encuentra todo lo referido al desarrollo del programa: Modulos, imagenes, bases de datos, instrucciones, etcétera.
##################################
ForUsers:
Carpeta donde se encuentra todo lo referente al uso por usuarios.
Intrucciones de como descargar y usar el programa, manual de usuario, y el ejecutable dentro de la carpeta: HealthApp.
##################################
Memorias:
Documentación tecnica del desarrollo del proyecto, incluye memorias, anexos y recursos utilizados para la creación de estos.
##################################
Poster:
Relativo a la representación visual del proyecto, usado para la descripción del proyecto a la hora de exponerlo.
##################################
Videos:
Ayudas audivisuales para el uso del programa y para el desarrollo de este. 
Se explica: Como instalar los requisitos para trabajar con el proyecto, como usarlo, como estan estructurados los datos, como funciona la formula de recomendación.
